### How to run

#### Database
- Create a user of following credentials
    - password: password
    - username: root
   
#### Spring Boot Application
- git clone https://github.com/UbaidurRehman1/animated-fortnight
- cd animated-fortnight
- cd GuestTracker
- mvn spring-boot:run
